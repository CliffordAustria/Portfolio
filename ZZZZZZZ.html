COLOR THEORY AND TYPOGRAPHY

Color theory is the set of rules and instructions that programmers use of visual interfaces to interact with users by pleasing color schemes. Designers use a color wheel to select the right colors each time and appeal to vast collected knowledge of human optical ability, psychology, history and more. Typography, layout or collection of letter types to be arranged into words and phrases to be disposed of when written on a page in blocks of script.

Activity 6 deals with color and typography games, which will allow one to get more acquainted with the various typography and color styles.
